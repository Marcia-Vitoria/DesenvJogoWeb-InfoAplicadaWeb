# Projeto: Desenvolvimento de Jogos Web

Projeto desenvolvido para a disciplina **Informática aplicada à Web**, no 3º ano do curso Técnico Integrado em Informática do **IFPB**. Professor Orientador: **José Neto**

## 1.  Sobre o projeto

O Alien Assault é um jogo web estático desenvolvido com HTML, CSS e JavaScript, no estilo de jogo de tiro.
Nele, o jogador pode controlar uma *nave espacial* e deve sobreviver a uma *invasão alienígena*, destruindo inimigos, desviando de projéteis e coletando power-ups que o ajudarão durante a partida.

O jogo conta com:

- Sistema de pontuação
- Vidas do jogador
- Inimigos normais e inimigos atiradores
- Power-ups (vida extra, escudo e tiro duplo)
- Efeitos sonoros gerados via Web Audio API
- Tela inicial, tela do jogo e tela de game over

## 2. Tecnologias utilizadas

* **HTML5** (Semântico)
    * Canvas para renderização do jogo
* **CSS3**
    * Flexbox
    * Gradientes
    * Animações e transições
    * Interface sci-fi estilizada
    * Media Queries (Responsividade)
* **JavaScript**
    * Manipulação do Canvas
    * Programação orientada a objetos
    * Detecção de colisões
    * Loop de animação
    * Gerenciamento de estados do jogo
* **Web Audio API**
     * Geração de efeitos sonoros sem arquivos externos

## 3. Como rodar localmente

Este é um projeto estático. Não há necessidade de instalação. Siga as orientações abaixo para roda-ló:

1.  Clone o repositório:
    ```sh
    git clone https://github.com/Marcia-Vitoria/DesenvJogoWeb-InfoAplicadaWeb.git
    ```
2.  Navegue até a pasta do projeto:
    ```sh
    cd DesenvJogoWeb-InfoAplicadaWeb
    ```
3.  Abra o arquivo `index.html` diretamente no seu navegador.
